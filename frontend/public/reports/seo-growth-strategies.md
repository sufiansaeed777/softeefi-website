## Foundation: Technical SEO

### Site Speed Optimization
1. **Core Web Vitals**
   - LCP < 2.5s
   - FID < 100ms
   - CLS < 0.1

2. **Implementation**
   - Image optimization (WebP format)
   - Lazy loading
   - CDN implementation
   - Minify CSS/JS

### Mobile-First Design
- Responsive design mandatory
- Mobile page speed priority
- Touch-friendly elements
- AMP for news sites

### Site Architecture
```
Homepage
├── Category Pages (10-15)
│   ├── Subcategory Pages
│   │   └── Product/Article Pages
└── Supporting Pages
    ├── About
    ├── Contact
    └── Resources
```

### Technical Checklist
- [ ] XML sitemap submitted
- [ ] Robots.txt optimized
- [ ] Schema markup implemented
- [ ] HTTPS enabled
- [ ] Canonical tags set
- [ ] 404 errors fixed

## Keyword Research Strategy

### The 70/20/10 Rule
- **70% Low competition** (0-30 difficulty)
- **20% Medium competition** (31-60 difficulty)
- **10% High competition** (61-100 difficulty)

### Keyword Types to Target

1. **Commercial Intent**
   - "best [product] 2024"
   - "[product] reviews"
   - "[product] vs [competitor]"
   - "buy [product] online"

2. **Informational Keywords**
   - "how to [solve problem]"
   - "what is [topic]"
   - "[topic] guide"
   - "[topic] tutorial"

3. **Local Keywords**
   - "[service] near me"
   - "[service] in [city]"
   - "best [service] [city]"

### Tools & Techniques
- Google Keyword Planner (free)
- Answer The Public
- Competitor gap analysis
- Search intent matching

## Content Strategy

### The Hub & Spoke Model

**Hub Page** (Pillar Content)
- 3,000+ words
- Comprehensive guide
- Links to all spokes

**Spoke Pages** (Supporting Content)
- 1,000-1,500 words
- Specific subtopics
- Link back to hub

### Content Calendar

Month 1-3: Foundation

Month 4-6: Scaling

## Link Building Tactics

### White Hat Strategies That Work

1. **Guest Posting**
   - Target DR 40+ sites
   - Relevant niches only
   - 2-3 per month
   - Author bio links

2. **HARO (Help a Reporter Out)**
   - Daily responses
   - Expert quotes
   - High-authority links
   - Brand mentions

3. **Broken Link Building**
   - Find broken links
   - Create better content
   - Reach out to webmasters
   - 15-20% success rate

4. **Resource Page Links**
   - Find resource pages
   - Create link-worthy content
   - Personalized outreach
   - 10-15% success rate

### Link Velocity
- Month 1-3: 5-10 links/month
- Month 4-6: 10-20 links/month
- Month 7-12: 20-40 links/month
